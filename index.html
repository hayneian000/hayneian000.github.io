This innovative forecasting system has significantly boosted revenue by 20% and simultaneously reduced waste by accurately predicting the demand for kitchen products across Maverik stores.

In addition to this, I have developed proprietary software applications focused on logistics and big data. These applications, crafted using my expertise in Angular, Spring Boot, and Java, have been instrumental in streamlining fuel logistics and other business processes, thereby effectively reducing operational costs.

My responsibilities also extended to creating high-performance Spring Boot Java applications. These were crucial for facilitating seamless data transfer between web applications and third-party software providers. This involved integrating Cron jobs, API gateways, and SQL to ensure efficient and reliable data handling.
